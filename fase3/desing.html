ELEMENTOS HTML

HTML es un markuo language, esto quiere decir que está escrito con código que se puede leer si que sea necesario en compilarlo primero, esto quiere decir que el texto de una página web este marcado que da instrucciones al navegador web como se tiene que mostrar el texto, estas son considerado como etiquetas de marcado de HTML. Estas escribiendo etiquetas HTML, esta hechas con un numero de partes específicas como: el carácter “menor que <”, el carácter “mayor que >”, etc.

¿INTRODUCCIÓN DE HTML Y ETIQUETAS?

HTML es el lenguaje para crear Sitios web. Un código escrito en este lenguaje es básicamente, un texto que el navegador mostrará en formato página web. El HTML es un lenguaje que se utiliza para la creación de las paginas web. Es un código que está escrito es un lenguaje básico el cual un texto que los navegadores (Firefox, Google Chrome, Opera, etc), lo muestra en un formato de pagina web. Estos textos que se genera el color, tamaño y fuente de letra, imágenes, fondo de imágenes, hiperenlaces y entradas de datos, entradas de datos, también cuneta con listas de selección, botones, etc., los cuales esta determinados y configurados mediante los indicadores llamados tasg. Un identificador o tags este permite fijar los atributos de tamaño, posición, imágenes de la pagina web. Estos identificadores constan de la apertura las cuales se establecieron las características y un cierre este cuando debe de dejar de hacer los efectos, restablecer a las características originales.
