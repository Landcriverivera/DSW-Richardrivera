ELEMENTOS HTML

HTML es un markuo language, esto quiere decir que está escrito con código que se puede leer si que sea necesario en compilarlo primero, esto quiere decir que el texto de una página web este marcado que da instrucciones al navegador web como se tiene que mostrar el texto, estas son considerado como etiquetas de marcado de HTML. Estas escribiendo etiquetas HTML, esta hechas con un numero de partes específicas como: el carácter “menor que <”, el carácter “mayor que >”, etc.
