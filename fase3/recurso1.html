¿COMO FUNCIONAN LAS ETIQUETAS HTML?


Las paginas web contiene una serie de elementos HTML esta contiene elementos diferentes etiquetas y atributos. Los elementos HTML son los componentes de una página web, por lo cual las etiquetas contienen la información y le indica al navegador de donde comienza y donde termina cada elemento y los atributos son la descripción de las características.


Estos elementos se dividen en tres partes, con la combinación de estas se crea el elemento HTML.


Etiquetas de apertura: Esta indica donde se comienza el elemento, esta esta conformadas o envueltas por corchete de apertura y cierre, como ejemplo se utiliza la etiqueta de inicio esta es para la creación de un párrafo.


Contenido: Es los que se puede ver o el resultado que ve la audiencia.
