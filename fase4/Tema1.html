ETIQUETAS BÁSICAS DE HTML

Etiquetas HTML y Etiquetas HTML es un markuo language, esto quiere decir que está escrito con código que se puede leer si que sea necesario en compilarlo primero, esto quiere decir que el texto de una página web este marcado que da instrucciones al navegador web como se tiene que mostrar el texto, estas son considerado como etiquetas de marcado de HTML. Estas escribiendo etiquetas HTML, esta hechas con un numero de partes específicas como: el carácter “menor que <”, el carácter “mayor que >”, etc. El hipertexto (HTML) este es un lenguaje informático que esta conformadas las paginas web y plataformas online, este no se puede considerar un lenguaje de programación, esta no pude crear funciones dinámicas, los usuarios web se crea la estructura sección, párrafo y enlaces que están conformados por elementos, etiquetas y atributos, están conformadas 142 etiquetas HTML disponible.

ELEMENTOS HTML

Los elementos HTML se dividen de tres partes, con la combinación de estas tres se crea un elemento HTML: Etiqueta de apertura: Esta cataloga o indica donde comienza el elemento, que se encuentran en corchetes de apertura y cierre para crear un párrafo. Contenido: es el contenido que ve el espectador. Etiqueta de cierre: Este es misma etiqueta de apertura con la diferencia con una barra inclinada delante del nombre del elemento para culminar el párrafo.

¿QUÉ SON LAS ETIQUETAS HTML?

Las etiquetas HTML son indicadores del lenguaje de marcas de hipertexto (o HTML, por sus siglas en inglés) que le dan formato y estructura al contenido de las páginas web.


Las etiquetas THML son estándares que adoptan los desarrolladores por medio de las etiquetas de texto, estos permiten a los navegadores interpretar los elementos de una página web sin importar donde fue creada o el tipo de idioma, por lo cual se puede mostrar al usuario. Esto permite las etiquetas HTML determine como se vera el sitio web, cada una de esta indica y representa un atributo que los navegadores deben interpretar y poder mostrarlo cuando el usuario los visite por internet.



COMO FUNCIONA LAS ETIQUETAS HTML

Por medio de las etiquetas de HTML, para esto la construcción de una página web puede ser entendida por la maquinas y los usuarios, mediante al código se basa la descripción de los elementos que la componen. También tiene una característica que es sencillo de aprender y accesible ya que cualquiera lo puede realizar una pagina web para la insertar imágenes, videos, enlaces. Cuando tiene acceso a la estructura de la página web puede reconocer las etiquetas de HTML, estas están escritas entre los signos: texto. La primera parte encerrada entre signos es la apertura de la etiqueta, y la segunda, que cuenta con una diagonal, indica el cierre de esa etiqueta, esto sirve para indicarle al navegador la estructura del texto indicada por dicha indicación. Para las paginas web se indica desde el comienzo del lenguaje la primera etiqueta o inicio , al pie de pagina se muestra el cierre por .
