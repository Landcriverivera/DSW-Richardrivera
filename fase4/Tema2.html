Elementos HTML

Los elementos HTML se dividen de tres partes, con la combinación de estas tres se crea un elemento HTML: Etiqueta de apertura: Esta cataloga o indica donde comienza el elemento, que se encuentran en corchetes de apertura y cierre para crear un párrafo. Contenido: es el contenido que ve el espectador. Etiqueta de cierre: Este es misma etiqueta de apertura con la diferencia con una barra inclinada delante del nombre del elemento para culminar el párrafo.

ATRIBUTOS HTML

Los atributos proporcionan información adicional sobre elementos HTML, todos los elementos HTML pueden tener atributos, estos atributos proporcionan información adicional sobre un elemento. Los atributos siempre se especifican en la etiqueta de inicio y los atributos generalmente vienen en pares nombre / valor como: nombre = "valor"


Fundamental de los elementos HTML son los atributos y cuentan con dos secciones:


  • Nombre: Esta es la identifica la información adicional que quiere agregar.


  • Valor del atributo: esta da mas detalles que el anterior.


    • DIFERENTES ATRIBUTOS

    El atributo href

    Los enlaces HTML se definen con la etiqueta . La dirección del enlace se especifica en el atributo href.


    El atributo src

    Las imágenes HTML se definen con la etiqueta , se busca la la ruta de dónde se encuentra la imagen se especifica en el atributo src.


    Los atributos width y height

    Las imágenes HTML también tienen atributos de ancho (width) y alto (height). El ancho y la altura se especifican en píxeles de forma predeterminada; entonces width= "100" significa 100 píxeles de ancho.


    El atributo alt

    El atributo alt especifica un texto que se utilizará, si no se puede mostrar una imagen. Los usuarios de pantalla pueden leer el valor del atributo alt. De esta manera, alguien "escucha" la página web, como una persona con problemas de discapacidad visual puede "escuchar" el elemento.


    El atributo style

    Se usa para especificar el estilo de un elemento, como color, fuente, tamaño, etc.


    El atributo lang

    El idioma del documento puede declararse en la etiqueta .


    El idioma se declara con el atributo lang.

    Se declarar un idioma es importante para las aplicaciones de accesibilidad como: lectores de pantalla y los motores de búsqueda.
